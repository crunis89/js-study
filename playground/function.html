함수

함수는 일련의 과정을 문(statement)들로 구현하고 코드 블록으로 감싸서 하나의 실행 단위로 정의한 것이다.

함수선언문

function add(x, y) { return x + y; }

런타임 이전에 함수 객체가 먼저 생성된다.

함수 선언문은 호이스팅에 영향을 받는다

함수 선언문은 표현식이 아닌 문이다.

함수 선언문은 “표현식이 아닌 문”

함수표현식

var add = function (x, y) { return x + y; };

함수 표현식은 호이스팅에 영향을 받지 않는다.

함수 표현식은 “표현식인 문”

함수 표현식으로 정의한 함수는 반드시 함수 표현식 이후에 참조 또는 호출하여야 한다.

호이스팅으로 인해 함수표현식이 권장된다.

재귀함수

재귀 함수(recursive function)는 자기 자신을 호출하는 행위, 즉 재귀 호출을 수행하는 함수를 말한다.

스코프

var x = 'global'; function foo() { var x = 'local'; console.log(x); // ① } foo(); console.log(x); // ②

바깥영역에 있는 변수x는 어디서든 참조 가능

함수 foo 내부에서 선언된 변수 x는 함수 foo 내부에서만 참조가능 하지만 함수 foo 외부에서는 참조 불가능

스코프내에서 var 키워드가 중복되면 변수가 재할당되어 변경된다

스코프내에서 let , const 키워드는 중복선언을 허용하지 않으며 SyntaxError: Identifier 'x' has already been declared 등의 에러가 뜬다
