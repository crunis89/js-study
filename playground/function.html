함수

함수는 일련의 과정을 문(statement)들로 구현하고 코드 블록으로 감싸서 하나의 실행 단위로 정의한 것이다.

함수선언문

function add(x, y) { return x + y; }

런타임 이전에 함수 객체가 먼저 생성된다.

함수 선언문은 호이스팅에 영향을 받는다

함수 선언문은 표현식이 아닌 문이다.

함수 선언문은 “표현식이 아닌 문”

함수표현식

var add = function (x, y) { return x + y; };

함수 표현식은 호이스팅에 영향을 받지 않는다.

함수 표현식은 “표현식인 문”

함수 표현식으로 정의한 함수는 반드시 함수 표현식 이후에 참조 또는 호출하여야 한다.

호이스팅으로 인해 함수표현식이 권장된다.
